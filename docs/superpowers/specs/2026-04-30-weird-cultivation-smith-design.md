## §1　定位与一句话

### 一句话

> **《我在诡异修仙造兵器》** 是一款单机挂机游戏。你扮演一位失去记忆的诡秘修仙世界铁匠铺老板。你不记得自己是谁，但你的手记得怎么打铁。每一炉都是一次开奖；每一位推门而入的访客都是一个盲盒；每一件造出来的兵器都会带着自己的履历回到你的器谱里——直到某一天，你从那本谱子上认出了自己。

> **名字本身是产品决策**：长、口语、自带钩子，符合现代独立游戏标题党命名风格（参考《我在精神病院当医生》《了不起的修仙模拟器》）。这种"过于直白"的标题与"诡异修仙"的内容形成反差，是叙事调性的第一层铺垫。

### 类型与代入

- **类型**：单机挂机 / 造装备模拟 / 诡秘氛围叙事
- **代入单位**：**铁匠铺老板**（个人尺度，**不是门派掌门**）
- **视角**：**2.5D 俯视**（铺子鸟瞰，可见 4 区域布局；老铁化身为白发老者剪影在铺中走动；离线回放叠加为"小本"窗口）
- **平台**：PC（Windows 优先），单机为主，**选配 Steam Cloud 存档同步**
- **会话目标**：单次 5-30 分钟，每日 1-2 次

### 核心爽点（按优先级）

1. **开炉时刻**——金光/异象/巧成的视觉反馈（老虎机几何学的"摇杆下拉")
2. **门铃响起**——盲盒访客的身份悬念
3. **回信归来**——装备带着履历/缺损/异变回到铺子（二次开奖）
4. **共鸣激活**——古谱凑齐瞬间的星图全亮
5. **身份微露**——主线暗线的某一段"老铁是谁"碎片

### 调性参考

- **氛围**：诡秘之主 + SCP + 国画水墨意境 + 老电影《摆渡人》般的"小铺子+大未知"
- **美术风格**：**国画风**（散点透视、留白、墨色为主、点染设色）。参考：宋代界画的建筑结构 + 明清版画的市井人物 + 当代水墨独立游戏《Inked》《Misty Continent》。**避免**日式像素风、卡通赛璐璐、写实 3D
- **质感参考**：《Recettear》的小店循环 × 《Disco Elysium》的氛围密度 × 《Moonlighter》白天造装备的体验
- **关键反例**：不要 AFK Arena 的"管理者+数字飞涨"，不要最强蜗牛的"杂烩百宝箱"

```
  · 离线: 系统按"铺规"自动跑                                     │
        │                                                      │
        ▼                                                      │

        │                                                      │
        ▼                                                      │
[③ 入器谱]  ── 装备落入对应 28 宿星位 / 玩家自连支脉 ─ 共鸣检定  │
        │                                                      │
        ▼                                                      │
[④ 问道门客]  ── 访客敲门 · 借/买 ── 盲盒身份 · 高酬深夜客       │

        ▼                                                      │
[⑤ 回信结算（二次开奖）]  ── 装备生履历 / 缺损 / 异变 / 不归还   │
        │                                                      │
        ▼                                                      │
[⑥ 履历回流]  ── 履历进星位档案 ── 名望/灵石/材料 回流 ──────────┘
```

### 三个开奖时刻（爽点节奏）

| # | 开奖 | 频率 | 期望强度 |
|---|---|---|---|
| A | **开炉出品质** | 高频（30s/在线; 2h/离线） | 中等爆点（金光） |
| B | **共鸣激活/反噬** | 低频（数日一次） | 大爆点（全谱亮） |
| C | **回信结算** | 中频（5-15min/在线; 1-3h/离线） | 中-大爆点（装备带故事回） |

> **三个时刻交错排布是这游戏的灵魂**。任何时候都不能让玩家"什么都没发生"地等超过 30 秒。

## §3　铁匠铺结构

铺子是固定 4 区域（不可扩展为门派规模），每区域可分阶升级 3 阶。

### 3.1 四区域

| 区域 | 功能 | 升级影响 |
|---|---|---|
| **炉房** | 开炉/淬火 | 单炉容量（同时几座炉）、巧成率窗口宽度、可炼品质上限（Lv.1 法 / Lv.2 禁 / Lv.3 秘）|
| **柜台** | 接客/出借/估价 | 客人座席数（同时可应几位）、估价精度、铺规槽数 |
| **阁楼** | 藏器/查谱 | 器谱可视化深度、装备档案保存条数、自连星轨笔数 |
| **后院** | 晾料/储料/养灰 | 材料栏位、特殊材料培育（如"养灰" "养异种料"） |

### 3.2 升级机制

- 每区域 3 阶升级，需 **灵石 + 名望阈值 + 特定材料**
- 升级**不解锁新系统**，只放大现有系统的容量/精度
- 升级后**不可降级**（避免博彩心理操控滥用）

### 3.3 老铁本人

- 1 名 NPC（玩家化身），无属性数值
- "手感"=玩家的实际操作熟练度（在线开炉时间窗口越准，巧成率+5%）
- 后院偶尔自言自语 1 句，成为**诡异叙事的注入点之一**

### 3.4 不要做的事

- ❌ 不做"学徒/小工"养成系统（违背降级原则）
- ❌ 不做铺子搬迁/扩建到第二家分店
- ❌ 不做铺子内部的装饰/家具自定义
- ❌ 不做炉子/柜台的多种类型选择（一座炉 = 一座炉，靠升级，不靠分支）

## §4　锻造（开奖）

### 4.1 锻造流程

```
[选配方] → [投料] → [生火] → [捶打/淬火] → [出炉]
                          ↑           ↑
                       倒计时窗口    开奖动画
```

- **配方**：玩家解锁的图谱中已知的兵器型号（开局 3 个，按器谱进度解锁）
- **投料**：每个配方有"必要料"+"可选添料"。添料可主动微调出品偏向（+巧成 / +某词缀偏好 / -反噬）
- **倒计时窗口**：在线时有一个 ~3 秒的"火候判定"窗口，玩家点击/松手时机影响巧成率
  - 离线时跳过此窗口，按基准巧成率计算

### 4.2 品质分布（基准）

| 品质 | 基准概率 | 巧成升阶后 | 视觉反馈 |
|---|---|---|---|
| 凡（白） | 60% | — | 普通铛声 |
| 灵（绿） | 25% | 凡 → 灵 +10% | 绿光 + 余烟 |
| 法（蓝） | 10% | 灵 → 法 +6% | 蓝光 + 哨声 |
| 禁（紫） | 4% | 法 → 禁 +2% | 紫光 + 屋震 |
| 秘（金） | **1%** | 禁 → 秘 +0.5% | **金光满屋 + 铺外起雾 + 自动写一段日记** |

### 4.3 巧成与反噬

- **巧成**：火候窗口准 / 添料配伍正确 / 老铁"今日手感"加成（每日开局随机 ±5%）
  - 巧成 = 该炉品质升一阶，**不可叠加**
- **反噬**（炉中物自燃 / 异变）：5% 基准，禁/秘料堆叠时升至 10%
  - 反噬产物：丢失原料，**留下一钵"灰"或一截"异种料"**——可作为下次添料的"赌注材料"
- **诡异变出**：极小概率（~0.1%）出炉得到一件**配方表里没有的兵器**——直接进"诡器谱"

### 4.4 装备数据结构（草案）

```
Equipment:
  id: 唯一 ID
  template_id: 配方型号
  quality: 凡|灵|法|禁|秘|诡
  affixes: [Affix, ...]  # 词缀列表
  origin: { date, ingredients, smith_state }  # 出生记录
  history: [HistoryEntry, ...]  # 履历，回信时追加
  star_position: { gu_id, su_id }  # 入谱星位
  status: 在铺|借出|不归还|损|异变
```

### 4.5 词缀（沿用现有 DESIGN.md §5 的方向）

- 凡缀：纯数值（攻+X、暴+X%）
- 灵缀：条件触发（夜战 +X / 对妖+X）
- 法缀：复合条件 + 小反噬（攻+X 但持有者每战自伤 1）
- **禁缀**：强力 + 强制反噬（攻+30% 但每战道心-5）
- **秘缀**：传说级 + 必须"封印"才能装备（消耗持有者道心上限）
- **诡缀**（新）：未知效果，仅在使用后通过回信揭示

> 词缀总池目标 60 条（凡15/灵15/法15/禁10/秘5），诡缀 ~10 条单独池。
> 当前 .tres 已有 21 条，需要补充 39 条 + 重新分类。

## §5　器谱（28 宿 + 自连）

### 5.1 三层结构

| 层 | 内容 | 可视化 |
|---|---|---|
| **外层** | 28 宿固定星位 + 主脉骨架（古谱预设） | 星图视图，固定布局，不可变 |
| **中层** | 每颗已点亮星位 = 一族装备的"祖星" | 点开 → 该族装备列表（带履历缩略） |
| **缝隙** | 玩家可用"星轨笔"在已点亮星位之间画自连 | 自连线为青色虚线，区别于金色主脉 |

### 5.2 古谱（首版 7 张）

每张古谱 = 28 颗星位，玩家可切换主图谱视图。首版 7 张：

| 古谱 | 主题 | 共鸣效果（凑齐） |
|---|---|---|
| 青龙宿 | 剑系兵器 | 出借兵器斩妖时回信故事强度 +1 级 |
| 玄武宿 | 防具/护符 | 借出兵器损坏概率 -50% |
| 朱雀宿 | 火/咒 | 反噬"灰"产出 +100% |
| 白虎宿 | 兽皮/骨器 | 怪客好感 +20% |
| 紫微宿 | 高品质（禁/秘） | 秘品出炉概率 ×1.5 |
| 血曜宿 | 邪/血 | 解锁"血祭"配方，深夜客 ×2 |
| 残宿 | 不归还的装备 | 不归还装备多年后回流概率 +30% |

### 5.3 入谱规则

- 每件装备出炉时，根据 `(配方型号, 品质, 主词缀)` 算出落点（确定性公式，不随机）
- 落点对应**当前选中古谱**的某颗星位
- 同一星位可叠加无限件装备（每件占 1 行履历）
- 星位"亮度"=该位上装备的数量与品质加权（仅视觉，不影响数值）

### 5.4 自连机制（支脉）

- 玩家通过下列途径获得**"星轨笔"**（消耗品，稀缺）：
  - 客人立大功归还时小概率留赠（~10%）
  - 反噬产物"灰" × 5 可炼一支
  - 共鸣激活时一次性奖励 1 支
  - 完成一张古谱时奖励 3 支
- 一支星轨笔 = 一条自连线，可在两颗已点亮星位之间画
- **隐藏图案表**（设计阶段秘密，玩家自己探索）：
  - 命中已知古图案 → 触发"共鸣暴击"（额外永久 buff，如所有锻造灵气消耗 -10%）
  - 命中"邪图案" → 触发"反噬"（永久负 buff，可解但代价大）
  - 未命中 → 仅作视觉装饰，无效果（但浪费了一支笔）

### 5.5 999 诡器谱（长期目标）

- 与 7 张古谱并行的另一本谱
- 目标：见过 999 件不同 `template_id × quality × 主词缀` 组合的装备
- 完成度奖励：每 100 件解锁一段"老铁身份碎片"

## §6　问道门客（盲盒）

### 6.1 客人三类

| 类 | 频次占比 | 身份 | 行为 |
|---|---|---|---|
| **常客** | 60% | 已知名（苏家娘子/小孟跑料/赵捕快…） | 借/买已知品类，回信稳定 |
| **罕客** | 30% | 来自远地 / 高门 | 出价高、品类挑、回信带故事 |
| **怪客** | 10% | **盲盒**（蒙面 / 无姓 / 来源不明） | 高酬高风险，可能不归还/带异变回 |

### 6.2 来访节奏

- **在线**：每 5-15 分钟一位（泊松分布）
- **离线**：每 1-3 小时一位
- **深夜（子-寅时）**：怪客占比 ↑ 至 30%

### 6.3 借/卖判定

每位客人到来时给出：
- **诉求**：求借 / 求购 一件兵器，附带模糊的需求描述（"能斩夜行物" / "压得住怨气"）
- **酬金**：灵石数 + 可能的稀缺料子
- **可见信息**：流派亲和（剑/咒/傀/丹/食/卜）、品阶感（"看着像灵 / 法品的人"）
- **不可见信息**（怪客特别）：真实身份、真实意图

玩家决策：
- **挑一件给他**：从器谱里挑（按词缀/品质/历史搜索）
- **拒绝**：-1 名望，但保留兵器
- **打听**：消耗一次"打听"道具，揭示部分隐藏信息

### 6.4 回信结算

借出后系统模拟一段"任务时间"（5min - 24h，按客人等级），到时回信。

| 结果 | 常客 | 罕客 | 怪客 | 装备影响 |
|---|---|---|---|---|
| 顺利归还 | 70% | 55% | 25% | +1 条履历，可能 +1 词缀（少见） |
| 立大功归还 | 12% | 20% | 25% | +1 条传奇履历，可能 +秘缀，名望大涨 |
| 损坏归还 | 10% | 12% | 15% | -1 词缀，需修复（消耗料） |
| 异变归还 | 5% | 8% | 20% | 词缀变诡，可能解锁新配方 |
| **不归还** | 3% | 5% | 15% | 进入"流失册"——可能多年后另一位客人带回 |

> 不归还的装备永远进**残宿**星位（参照 §5.2）。
> 同一品质装备 + 同一类型客人，结果分布是上表的固定权重，**不受名望/铺子等级影响**——这是博彩学的"基础概率不可被腐化"原则。

### 6.5 名望与铺子声誉

- 累积名望 → 解锁高阶客人 / 高阶配方 / 升级铺子
- 名望可掉（拒客 / 装备坑客 / 反噬客人导致 NPC 死亡）
- 名望高 = 来料商人折扣 + 更高酬金客人占比

## §7　铺规与时间线

### 7.1 时间线模型

- 游戏世界以"时辰"（=2 小时）为最小推进单位
- 在线/离线**产出公式相同**，只是离线由系统按铺规跑
- 离线时长无封顶（与原 DESIGN.md 的 12h 软帽不同）
- **"老铁打盹"机制**：当**单次离线**超过 24 小时，超过部分按 70% 衰减结算；超过 72 小时，按 30% 结算。叙事呈现："老铁趴在炉边睡过去了，错过了一些事。"——是衰减不是惩罚，且只对**单次离线**计算，玩家上线后清零。

### 7.2 铺规可设项（开局 3 槽，柜台升级到 +5 槽）

| 槽类 | 示例 |
|---|---|
| **接客规** | "深夜 + 高酬 → 借" / "怪客一律拒" / "剑系常客 → 借灵品以下" |
| **开炉规** | "灵气 ≥ 200 → 自动开炉灵·剑配方" |
| **应料规** | "玄铁 ≥ 5 → 自动收购至上限" |

### 7.3 铺规可被攻破

- 怪客可"伪装"成符合铺规的客人（穿常客的衣服）
- 玩家在线时收到"识破检定"提示，离线时按铺规自动放行（**这是离线诡异感的最大注入点**）
- 攻破后玩家可学习新铺规条款（如"鞋底无尘 = 怪客 → 拒"），系统永久解锁

### 7.4 在线 vs 离线的本质区别

| 维度 | 在线 | 离线 |
|---|---|---|
| 产出量 | 一致 | 一致 |
| 决策权 | 玩家逐一拍板 | 系统按铺规自动 |
| 巧成率 | 玩家手感 +0~5% | 基准值 |
| 怪客识破 | 玩家可识破 | 按铺规判定 |
| 共鸣激活时刻 | 玩家亲见 | 进日记，回放时看 |

## §8　世面（势力背景板）

### 8.1 6 个势力（玩家不操作）

| 势力 | 风格 | 关系基线 | 影响维度 |
|---|---|---|---|
| 问道宗 | 名门正派、大户 | 中-友 | 来料/来单/常客 |
| 寒星宿宗 | 北地世家、富贵 | 中-友 | 高酬客 |
| 枯荣谷 | 半正半邪、商道 | 中 | 主要供料商 |
| 万丹门 | 丹道一脉、清贫 | 中 | 罕客 |
| 雾丘野修 | 散修、亡命 | 中-敌 | 怪客 |
| ?（未知名） | 邪修传闻 | 敌 | 深夜怪客 + 神秘料 |

### 8.2 影响机制

每势力有一个"动态状态"（如"问道宗近月外勤多"），由系统脚本随机推进（每周轮换 2-3 个）：
- 影响**当前可来什么客**（来客池权重）
- 影响**当前可收什么料**（来料池权重）
- 影响**当前可接什么单**（公榜配方需求）

### 8.3 不要做的事

- ❌ 不做势力间战争 / 联姻 / 贸易系统
- ❌ 不做势力关系经营（玩家不能"主动改善关系"）
- ❌ 不做势力大事件 BOSS

> 势力 = **天气**。它影响你的生意，但你不能下雨。

## §9　诡异叙事生成

### 9.1 三类叙事来源

| 类 | 形式 | 产出方式 |
|---|---|---|
| **手写卡片** | 关键事件触发的固定文本 | 设计阶段全部手写（~150 张） |
| **程序化拼装** | 离线日记段落 | 模板 + 占位符（客人名/装备名/时辰） |
| **暗线碎片** | 老铁身份相关 | 手写 + 触发条件（每 100 件诡器谱解锁一段） |

### 9.2 手写卡片库目标（首版）

| 类目 | 数量 |
|---|---|
| 客人首次到访 | 30 |
| 怪客离奇行为 | 25 |
| 反噬异象 | 20 |
| 巧成/秘品出炉 | 15 |
| 共鸣激活 | 15 |
| 不归还回流 | 10 |
| 老铁自言自语 | 20 |
| 暗线碎片（老铁身份） | 15 |

### 9.3 程序化日记段落（每次离线后）

模板示例：
```
[时辰]，门铃响过[次数]次。
[客人描述]推门而入，[要求描述]。
按铺规，[处理结果]。
[诡异细节，如有]。
```

模板池 ~30 个，组合后能产生 ~10000 种文字组合，避免"机器感"。

### 9.4 老铁身份暗线（长尾主线）

- 玩家不主动追，系统不强推
- 通过：
  - 999 诡器谱进度（每 100 件解锁 1 段）
  - 特定怪客（认得老铁的人）
  - 后院的几件"不知何来的器物"（每件需要凑齐特定古谱才能"看清"）
- 终局：揭示老铁身份后**不强制结束游戏**，玩家可继续编谱

## §10　数值与博彩学

### 10.1 关键参数表

| 参数 | 默认值 | 说明 |
|---|---|---|
| 单炉时间（在线） | 30 秒 | 含投料+生火+捶打+出炉动画 |
| 单炉时间（离线） | 2 小时 | 系统自动 |
| 在线开炉巧成率窗口 | 3 秒 | 火候判定 |
| 客人来访间隔（在线） | 5-15 分钟 | 泊松，λ=10 min |
| 客人来访间隔（离线） | 1-3 小时 | 泊松，λ=2 h |
| 灵石每客平均产出 | 100 / 300 / 800 | 常客 / 罕客 / 怪客（基础酬金，立大功额外 ×3） |
| 名望升铺阈值 | 100 / 500 / 2000 | 三阶 |
| 28 宿凑齐预期时长 | ~7-14 天 | 第一张古谱 |
| 999 诡器谱预期时长 | ~6-12 月 | 长尾 |

### 10.2 7 日玩家旅程

| 日 | 关键体验 | 留人钩子 |
|---|---|---|
| D1（30 min） | 5 炉、3 客、解锁 3 星位、首次设铺规 | 看到 28 宿全图，差 25 颗 |
| D2 | 第一次离线回看日记 | 看到自己离线时发生的故事 |
| D3 | 第一次出灵品+留赏 | 经济正循环启动 |
| D4 | 第一次反噬、第一次怪客 | 诡异感首次注入 |
| D5 | 第一张古谱差 3 颗 | 近失感最大化 |
| D6 | 凑齐青龙宿，第一次共鸣 | **首次大爆点** |
| D7 | 解锁第一段身份碎片 | 长尾钩子建立 |

### 10.3 老虎机几何学要求

- **任何 30 秒内必有一次微反馈**（铛声/光效/数字跳动）
- **任何 5 分钟内必有一次中爆点**（出灵品 / 客人到 / 回信归来）
- **任何 1 小时内必有一次大爆点**（出禁/秘 / 共鸣 / 怪客异变）
- 反之不允许（避免疯狂塞反馈造成审美疲劳）

### 10.4 反操控原则

- 不做"伪保底"假象（玩家信不过的保底等于没保底）
- **真保底**：连续 50 炉无禁/秘 → 第 51 炉强制出至少禁品（玩家不知道这条规则但能感受到）
- 不做日常任务、签到奖励、无聊扫荡——**只有铺子本身的事**
